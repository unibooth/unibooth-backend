# 특징

- Spring Boot를 활용한 백엔드 API 서버 개발
- Kakao Oauth를 활용한 유저 정보 수집 최소화 및 편의성 제공
- JWT 토큰 컨트롤로 보안성 극대화
- Static Factory Method를 통한 Data Encapsulization 및 이름 의미 부여
- Single responsibility principle를 준수하고자 노력
